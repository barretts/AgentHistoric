# Agent Historic GitHub Pages Site

This branch hosts the public project page for Agent Historic.

## Local Development

Prerequisites: Node.js 20+

1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`
3. Build static output: `npm run build`

## Deploy to GitHub Pages

1. Build the site: `npm run build`
2. Publish the generated `dist/` output using your preferred Pages workflow
3. Commit and push the `pages` branch

Repository: https://github.com/barretts/AgentHistoric
